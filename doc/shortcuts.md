
### CDPnP App Shortcuts

```
<-   : Move left     : Left arrow
->   : Move right    : Right arrow
^    : Move forward  : Up arrow
v    : Move backward : Down arrow
Up   : Move up       : ' (")
Down : Move down     : / (?)
CW   : CW rotate     : , (<)
CCW  : CCW rotate    : . (>)
       Set speed     : 1, 2, 3, 4
       Toggle pause  : Space
```
